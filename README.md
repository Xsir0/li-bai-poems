# 李白诗选 · 图文并茂

这是一个纯静态网页，选取四首李白短诗配以水墨风插画。

## 预览

在当前目录运行：

```bash
python -m http.server 8000
```

然后打开 `http://localhost:8000/` 查看。

## 目录结构

- `index.html`
- `styles.css`
- `assets/`

## 发布

可直接上传到任意静态托管服务（例如 GitHub Pages、Netlify、Vercel）。
